# Anomaly Detection App with Autoencoder

## Overview
This project implements an anomaly detection system using an Autoencoder neural network. It detects unusual patterns in data by learning to reconstruct normal data and flagging deviations as anomalies.

## Features
- Unsupervised anomaly detection using Autoencoder
- Easy-to-use interface (mention if you have a UI or API)
- Supports model saving and loading (.h5 files)
- Handles data uploads for detection
